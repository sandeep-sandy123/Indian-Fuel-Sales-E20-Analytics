# ⛽ Indian Fuel Sales & E20 Analytics Dashboard

## 📌 Project Overview

This project analyzes Indian fuel sales, station performance, customer behavior, and E20 fuel adoption using **SQL Server, Python, Excel, and Power BI**.

The objective is to transform raw transactional data into meaningful business insights through data cleaning, SQL analysis, interactive dashboards, and executive reporting.

---

# 🎯 Business Problem

Fuel companies need to understand:

- Which states and cities generate the highest revenue?
- Which fuel type contributes the most revenue?
- Are fuel stations achieving their monthly revenue targets?
- How is E20 fuel adoption performing?
- What trends can help improve business decisions?

This project answers these business questions using end-to-end data analytics.

# 💡 Key Business Insights

- Karnataka generated the highest fuel revenue.
- Petrol contributed the highest overall revenue.
- Bengaluru recorded the strongest sales performance.
- Commercial customers generated higher revenue than individual customers.
- Several stations exceeded their monthly revenue targets.
- E20 fuel adoption showed consistent growth across selected locations.
